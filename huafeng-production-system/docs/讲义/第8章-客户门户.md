## 8.1 客户门户概述

### 需求背景

华丰电梯的客户经常打电话询问订单进度，导致：
- 销售需要花大量时间回复客户
- 信息传递不透明，客户体验差
- 客户无法自助查看进度

### 功能需求

| 功能 | 说明 |
|-----|------|
| 合同号查询 | 输入合同号查看订单 |
| 订单列表 | 展示客户所有订单 |
| 进度详情 | 整体进度、工艺进度、零件进度 |
| 移动端适配 | 手机上方便查看 |

## 8.3 Material Design 风格

### 设计原则

1. **简洁** - 去除不必要的装饰
2. **层次** - 通过阴影体现层次
3. **一致** - 统一的视觉语言
4. **响应** - 动画反馈

### 样式实现

```css
/* 卡片 */
```

## 8.5 核心逻辑

### 进度计算

```javascript
const getProgress = (order) => {
  const procs = ['剪板', '数冲', '激光下料', '折弯', 
                 '角钢线', '切管机', '焊接', '喷涂', '组装']
  let completed = 0
  procs.forEach(p => {
    if (order.processes?.[p]?.status === '已完成') {
      completed++
    }
  })
  return Math.round((completed / procs.length) * 100)
}
```

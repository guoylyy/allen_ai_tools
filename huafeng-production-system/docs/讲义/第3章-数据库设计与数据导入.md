## 3.1 数据库设计基础

### 什么是数据库设计？

数据库设计是指根据业务需求，确定数据的存储结构、表之间的关系。

### 设计步骤

```
需求分析 → 概念设计 → 逻辑设计 → 物理设计 → 实现
```

### 核心原则

1. **一对一** - 一个订单对应一个客户
2. **一对多** - 一个订单包含多个产品
3. **多对多** - 一个工人可以负责多个工艺，一个工艺可以由多个工人负责

### 派工单结构

- 12个工序工作表
- 1个工艺流程卡

### 解析 Excel

#### 安装 xlsx

```bash
npm install xlsx
```

#### 解析代码

```javascript
import * as XLSX from 'xlsx'

/**
 * 解析 Excel 文件
 * @param {ArrayBuffer} data - Excel 文件数据
 * @returns {Object} 解析后的数据
 */
const parseExcel = (data) => {
  // 读取 workbook
  const workbook = XLSX.read(data, { type: 'array' })
  
  // 获取所有工作表
  const sheetNames = workbook.SheetNames
  
  // 存储结果
  const result = {
    orders: [],
    products: [],
    processes: []
  }
  
  // 遍历每个工作表
  sheetNames.forEach(sheetName => {
    if (sheetName === '工艺流程卡') {
      // 解析工艺流程卡
      const products = parseProcessCard(workbook.Sheets[sheetName])
      result.products = products
    } else {
      // 解析工序工作表
      const processData = parseProcessSheet(workbook.Sheets[sheetName])
      result.orders.push(...processData)
    }
  })
  
  return result
}
```

## 3.5 数据存储（SQL.js）

### 初始化数据库

```javascript
// server/db.js
const initSqlJs = require('sql.js')

let db = null

const initDB = async () => {
  const SQL = await initSqlJs()
  
  // 尝试加载已有数据
  const savedData = localStorage.getItem('huafeng_db')
  if (savedData) {
    const uint8Array = new Uint8Array(JSON.parse(savedData))
    db = new SQL.Database(uint8Array)
  } else {
    db = new SQL.Database()
  }
  
  // 创建表
  createTables()
  
  return db
}

const createTables = () => {
  // 订单表
  db.run(`
    CREATE TABLE IF NOT EXISTS orders (
      id TEXT PRIMARY KEY,
      customer TEXT,
      project TEXT,
      deliveryDate TEXT,
      quantity INTEGER,
      status TEXT,
      createdAt TEXT
    )
  `)
  
  // 产品表
  db.run(`
    CREATE TABLE IF NOT EXISTS products (
      sid TEXT PRIMARY KEY,
      orderId TEXT,
      name TEXT,
      partNo TEXT,
      processRoute TEXT,
      quantity INTEGER,
      completedProcs TEXT,
      FOREIGN KEY (orderId) REFERENCES orders(id)
    )
  `)
  
  // 更多表...
}

export { db, initDB }
```

### 数据持久化

```javascript
// 保存到 localStorage
const saveDB = () => {
  const data = db.export()
  const arr = Array.from(data)
  localStorage.setItem('huafeng_db', JSON.stringify(arr))
}

// 每次数据变更后保存
const saveData = () => {
  saveDB()
  // 同时保存到文件
  const fs = require('fs')
  const data = db.export()
  fs.writeFileSync('./db.sqlite', Buffer.from(data))
}
```

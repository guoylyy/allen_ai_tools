## 9.4 PM2 进程管理

### 安装 PM2

```bash
npm install -g pm2
```

### 启动服务

```bash
# 启动后端
pm2 start index.js --name huafeng-api

# 查看状态
pm2 status

# 查看日志
pm2 logs huafeng-api

# 重启
pm2 restart huafeng-api
```

### 常用命令

| 命令 | 说明 |
|-----|------|
| pm2 start | 启动服务 |
| pm2 stop | 停止服务 |
| pm2 restart | 重启服务 |
| pm2 logs | 查看日志 |
| pm2 monit | 监控面板 |
| pm2 save | 保存进程列表 |
| pm2 startup | 设置开机自启 |

### 开机自启

```bash
# 保存当前进程
pm2 save

# 生成开机自启脚本
pm2 startup

# 按照提示执行命令
```

## 9.5 Nginx 配置

### 安装 Nginx

```bash
# Ubuntu
sudo apt update
sudo apt install nginx

# 启动
sudo systemctl start nginx
sudo systemctl enable nginx
```

### 配置反向代理

```nginx
# /etc/nginx/sites-available/huafeng

server {
    listen 80;
    server_name huafeng.yourdomain.com;

    # 前端静态文件
    location / {
        root /var/www/huafeng/dist;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    # 后端 API
    location /api {
        proxy_pass http://127.0.0.1:3001;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_cache_bypass $http_upgrade;
    }
}
```

### 启用配置

```bash
# 创建链接
sudo ln -s /etc/nginx/sites-available/huafeng /etc/nginx/sites-enabled/

# 测试配置
sudo nginx -t

# 重启 Nginx
sudo systemctl restart nginx
```

## 9.6 SSL 证书（HTTPS）

### 使用 Let's Encrypt（免费）

```bash
# 安装 Certbot
sudo apt install certbot python3-certbot-nginx

# 获取证书
sudo certbot --nginx -d huafeng.yourdomain.com

# 自动续期
sudo certbot renew --dry-run
```

### 强制 HTTPS

```nginx
server {
    listen 80;
    server_name huafeng.yourdomain.com;
    return 301 https://$server_name$request_uri;
}
```

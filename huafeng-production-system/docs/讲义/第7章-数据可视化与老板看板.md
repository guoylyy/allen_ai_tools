## 7.1 数据可视化概述

### 什么是数据可视化？

将数据转换为图形，帮助人们更直观地理解数据。

### 常用图表类型

| 图表 | 适用场景 |
|-----|---------|
| 柱状图 | 对比不同类别的数据 |
| 折线图 | 展示趋势变化 |
| 饼图 | 展示占比 |
| 进度环 | 展示完成率 |
| 散点图 | 展示分布关系 |

---

## 7.2 ECharts 集成

### 安装

```bash
npm install echarts
```

### 基本使用

```javascript
import * as echarts from 'echarts'

const initChart = () => {
  const chartDom = document.getElementById('main')
  const myChart = echarts.init(chartDom)
  
  const option = {
    title: {
      text: '生产统计'
    },
    tooltip: {},
    xAxis: {
      data: ['剪板', '数冲', '折弯', '焊接']
    },
    yAxis: {},
    series: [{
      name: '产量',
      type: 'bar',
      data: [120, 200, 150, 80]
    }]
  }
  
  myChart.setOption(option)
}
```

## 7.4 图表实现

### 生产进度环

```javascript
const getProgressOption = (completed, total) => {
  const percent = Math.round((completed / total) * 100)
  
  return {
    series: [{
      type: 'gauge',
      startAngle: 180,
      endAngle: 0,
      min: 0,
      max: 100,
      splitNumber: 8,
      axisLine: {
        lineStyle: {
          width: 6,
          color: [[1, '#667eea']]
        }
      },
      pointer: {
        icon: 'path://M12.8,0.7l12,40.1H0.7L12.8,0.7z',
        length: '12%',
        width: 20,
        offsetCenter: [0, '-60%'],
        itemStyle: { color: '#00ff00' }
      },
      axisTick: { length: 12, lineStyle: { color: 'auto', width: 2 } },
      splitLine: { length: 20, lineStyle: { color: 'auto', width: 5 } },
      axisLabel: { color: '#464646', fontSize: 12 },
      title: { offsetCenter: [0, '-10%'], fontSize: 14 },
      detail: {
        fontSize: 30,
        offsetCenter: [0, '0%'],
        valueAnimation: true,
        formatter: '{value}%',
        color: '#00ff00'
      },
      data: [{ value: percent, name: '完成率' }]
    }]
  }
}
```

### 工艺产量柱状图

```javascript
const getProcessOption = (data) => {
  return {
    tooltip: { trigger: 'axis' },
    xAxis: {
      type: 'category',
      data: ['剪板', '数冲', '折弯', '焊接', '喷涂'],
      axisLabel: { color: '#666' }
    },
    yAxis: { type: 'value' },
    series: [{
      data: data,
      type: 'bar',
      itemStyle: {
        color: new echarts.graphic.LinearGradient(0, 0, 0, 1, [
          { offset: 0, color: '#667eea' },
          { offset: 1, color: '#764ba2' }
        ])
      }
    }]
  }
}
```

## 6.2 角色权限设计

### 角色说明

| 角色 | 权限 |
|-----|------|
| 普通工人 | 扫码报工 |
| 工艺组长 | 排单、报工审核、查看统计 |
| 管理员 | 全部功能 |

### 权限控制

```javascript
const hasPermission = (action) => {
  const role = currentUser.value?.role
  
  const permissions = {
    '普通工人': ['report'],
    '工艺组长': ['report', 'schedule', 'review', 'stats'],
    '管理员': ['report', 'schedule', 'review', 'stats', 'manage']
  }
  
  return permissions[role]?.includes(action) || false
}
```

## 6.4 CRUD 实现

### 新增员工

```javascript
const saveWorker = () => {
  if (editingWorker.value) {
    // 编辑
    const index = workers.value.findIndex(w => w.id === form.value.id)
    workers.value[index] = { ...form.value }
  } else {
    // 新增
    form.value.id = 'W' + String(workers.value.length + 1).padStart(3, '0')
    workers.value.push({ ...form.value })
  }
  
  showAddModal.value = false
}
```

### 删除员工

```javascript
const deleteWorker = (id) => {
  if (confirm('确定要删除该员工吗？')) {
    workers.value = workers.value.filter(w => w.id !== id)
  }
}
```

### 状态切换

```javascript
const toggleStatus = (worker) => {
  worker.status = worker.status === '在职' ? '离职' : '在职'
}
```

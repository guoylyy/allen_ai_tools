# 华丰电梯生产管理系统 - 上线准备清单

## 一、基础设施准备

### 1. 服务器
- [ ] **云服务器** (推荐配置)
  - 配置：2核4G内存 / 4核8G内存
  - 系统：Ubuntu 22.04 LTS 或 CentOS 8
  - 硬盘：50GB SSD 起
  - 带宽：5Mbps 起

- [ ] **域名**
  - 域名注册：如 `hf.abc.com` 或 `生产.华丰电梯.cn`
  - 完成 ICP备案（国内服务器必需）
  - SSL证书（HTTPS）

### 2. 数据库
- [ ] **MySQL** (推荐) 或 PostgreSQL
  - 版本：MySQL 8.0+ / PostgreSQL 14+
  - 字符集：utf8mb4
  - 备份策略：每日自动备份

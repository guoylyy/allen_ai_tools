#### 4.3 阻塞算法实现（核心难点）

**问题**：如何判断某个工艺是否可以排单？

**逻辑**：
```
A 工艺可以排单的条件：
1. 前面所有工序都已完成
2. 或者该工艺尚未开始
```

**实现代码**：
```javascript
const computeProcessStatus = (order) => {
  const procs = ['剪板', '数冲', '激光下料', ...]
  
  procs.forEach((proc, index) => {
    if (order.processes[proc]?.status === '已完成') return
    
    // 获取前置工艺
    const prevProcs = getPrevProcesses(proc)
    const allPrevCompleted = prevProcs.every(p => 
      order.processes[p]?.status === '已完成'
    )
    
    order.processes[proc].canSchedule = allPrevCompleted
    order.processes[proc].status = allPrevCompleted ? '可排单' : '已阻塞'
  })
}
```

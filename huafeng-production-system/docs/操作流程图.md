## 关键功能点

### ✅ 生产排单
- 按订单维度查看所有工序
- 按工艺维度查看所有订单
- 工序流程关系图展示
- 智能阻塞判断（上道工序未完成则无法排单）

### ✅ 扫码报工
- 选择工艺 → 选择订单 → 报工生产
- 实时完成进度
- 完成后自动流转到下游工序

### ✅ 员工管理
- 角色：普通工人、工艺组长
- 负责工艺多选
- 手机号、姓名、入职时间

### ✅ Boss看板
- 赛博朋克风格
- 实时生产数据展示

---

**系统优势：**
1. 直观的生产流程可视化
2. 工序依赖自动管理
3. 扫码报工实时准确
4. 多维度统计分析

# 太稳投资官网 - 设计规范

## 1. 项目概述
- **项目名称**：太稳投资官方网站
- **类型**：单页落地页
- **核心功能**：展示AI时代稳健投资机构的品牌形象与投资理念
- **目标用户**：寻找稳健投资渠道的高净值客户、企业决策者

## 2. UI/UX 规范

### 2.1 布局结构
- **导航栏**：固定顶部，透明背景，滚动后变为深色
- **首屏 (Hero)**：100vh 全屏，视频/渐变背景 + 大标题
- **投资理念区**：SVG 动画展示"曲折中的平稳"
- **机构介绍**：图文结合
- **投资案例**：卡片网格布局
- **联系方式**：三列信息展示
- **页脚**：版权信息

### 2.2 响应式断点
- 移动端：< 768px
- 平板：768px - 1024px
- 桌面：> 1024px

### 2.3 配色方案
```
--primary: #0F172A        /* 深海蓝 - 主色 */
--secondary: #D4AF37      /* 哑光金 - 辅助色 */
--accent: #0EA5E9        /* 科技青 - 强调色 */
--bg-light: #F8FAFC      /* 浅背景 */
--bg-white: #FFFFFF      /* 白色背景 */
--text-dark: #1E293B      /* 深色文字 */
--text-light: #64748B    /* 浅色文字 */
--text-white: #F8FAFC    /* 白色文字 */
```

### 2.4 字体
- **标题**：Noto Serif SC, serif
- **正文**：Inter, PingFang SC, sans-serif
- **字号**：
  - H1: 72px (桌面) / 40px (移动)
  - H2: 48px (桌面) / 32px (移动)
  - H3: 24px
  - Body: 16px
  - Small: 14px

### 2.5 间距系统
- Section 间距：120px (桌面) / 80px (移动)
- 容器最大宽度：1200px
- 栅格间隙：32px

### 2.6 动画效果
- **入场动画**：淡入上浮，duration 0.8s ease-out，stagger 延迟 0.1s
- **理念动画**：SVG 路径动画，圆点沿路径平滑移动
- **悬停效果**：transform scale(1.02), transition 0.3s

## 3. 功能规范

### 3.1 导航
- Logo 左侧
- 导航链接：理念、案例、联系
- 平滑滚动到对应区域
- 移动端：汉堡菜单

### 3.2 首屏
- 主标题："AI时代的静水流深"
- 副标题："在喧嚣中寻找确定性，于曲折中平稳前行"
- CTA 按钮："了解我们的策略" → 滚动到理念区

### 3.3 投资理念区
- SVG 曲线动画：蜿蜒线条 + 匀速移动的圆点
- 理念文字展示

### 3.4 机构介绍
- 核心文案："我们不是追逐风口的赌徒，而是AI浪潮中的摆渡人"
- 强调AI时代稳健投资的重要性

### 3.5 投资案例卡片
- 企业 Logo/图标
- 企业名称
- 投资领域
- 简述

### 3.6 联系方式
- 公司地址
- 商务邮箱
- 微信公众号二维码区域
